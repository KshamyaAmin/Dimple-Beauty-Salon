#Dimple Beauty Salon Website

A modern, elegant, and responsive website built for **Dimple Beauty Salon**, showcasing beauty services, gallery, and customer experience with a premium design.

---

## 🌐 Live Website

🔗 [https://dimple-beauty-salon-67lbo3xt5-kshamya-amins-projects.vercel.app/](https://dimple-beauty-salon-67lbo3xt5-kshamya-amins-projects.vercel.app/)

---

## ✨ Features

* 💅 Clean and aesthetic salon UI
* 📱 Fully responsive design
* 🖼️ Gallery section for showcasing work
* 📖 About section with brand story
* 💼 Services section (hair, skin, makeup, etc.)
* 📞 Contact section for easy reach
* ⚡ Smooth navigation and fast loading

---

## 🧠 Project Overview

This project is designed to create a **strong digital presence** for a beauty salon by combining elegant visuals with a user-friendly interface.

It highlights services such as:

* Hair styling & treatments
* Skin care & facials
* Bridal & party makeup
* Nail art and beauty services ([Dimple Beauty Salon][1])

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Dimple-Beauty-Salon.git
```

2. Open the project

```bash
cd Dimple-Beauty-Salon
```

3. Run locally

* Open `index.html` in your browser
